# Examples based goldfiles

This location contains the manifest files and related files generated by poppy for the [examples](../examples) directory tree.

This makes it easier to detect fundamental changes in how manifest files are generated that may not be caught by tests.

To regenerate the goldfiles, run from the root of this repository:

```
./tests/examples_goldfiles/run_goldfiles_test.sh 
```

If all is well, then there should be no "git diff".
